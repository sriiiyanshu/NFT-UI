## Design Features

All pages maintain consistent design:
- **Typography**: Clash Display for headings, Poppins for body text
- **Color Scheme**: Purple/pink gradient theme with dark background
- **Effects**: Glass morphism, backdrop blur, gradient overlays
- **Responsive**: Mobile-first design with breakpoints for tablet and desktop
- **Navigation**: Active state highlighting in navbar
- **Footer**: Consistent across all pages with social links

## Navigation

Navigate between pages using the top navigation bar:
- Logo (left) - Returns to home
- Menu items (right) - About, Features, Pricing, Contact
- Connect Wallet button (far right)

All pages are fully functional with React Router DOM.
